## 3. CÓDIGO DEL PROGRAMA
# Contexto: Script en Python para conectar con el computador cuántico (NMR).

### Estructura del Script (`nombre_app.py` o Jupyter Notebook)

```python
# IMPORTACIÓN DE MÓDULOS
# Se importan las herramientas necesarias del SDK spinqit
from spinqit import get_nmr, get_compiler, Circuit, NMRConfig
from spinqit import H, CX, Rx

# 1. CONFIGURACIÓN DEL MOTOR
# 'get_nmr()' inicializa el backend para Resonancia Magnética Nuclear
engine = get_nmr()
comp = get_compiler("native")

# 2. DEFINICIÓN DEL CIRCUITO
circ = Circuit()
# Se asignan 2 qubits (Límite físico del equipo NMR usado)
q = circ.allocateQubits(2)

# Aplicación de compuertas Hadamard (H) para superposición
circ << (H, q[0])
circ << (H, q[1])

# 3. COMPILACIÓN
# Se compila el circuito para el backend nativo
exe = comp.compile(circ, 0)

# 4. CONFIGURACIÓN DE LA CONEXIÓN (NMRConfig)
# Estos parámetros son específicos para el acceso remoto al hardware
config = NMRConfig()
config.configure_shots(1024)            # Número de repeticiones del experimento
config.configure_ip("172.27.52.229")    # Dirección IP del equipo cuántico
config.configure_port(8989)             # Puerto de comunicación
config.configure_account("SpinQ001", "123456") # Autenticación (Usuario, Pass)
config.configure_task("Test-JUT", "Test-JUT")  # Identificador de la tarea

# 5. EJECUCIÓN Y SALIDA
# Se envía la tarea al motor y se imprimen las probabilidades resultantes
result = engine.execute(exe, config)
print(result.probabilities)
